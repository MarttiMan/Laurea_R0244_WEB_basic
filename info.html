Create a Website (coding exercise)

Tämä tehtävä on suoritettu osanana Verkkosivujen kehittäminen R0244-3025 kurssia. Aiheeksi valikoitunut omat lapset.

Tehtävänanto

  1. Create a website with a topic and theme of your choice. Use html and css properties as versatile as possible. The website should also work with mobile devices. Make the website visually impressive. Name the homepage index.html.
  2. Make sure that users are able to navigate to every page in site and it's easy to use / clear.
  3. Validate your HTML and CSS code using https://validator.w3.org/ (Linkittyy ulkoiselle sivustolle.) and https://jigsaw.w3.org/css- validator/validator.html.en (Linkittyy ulkoiselle sivustolle.) (and fix if needed)
  4. Create a short report including:
    • Site subject / theme (why)
    • Site structure (how)
    • User media (pictures, sound, video...)
    • Special features (eg. the possible use of javascript or any other special technique)
    • The validation report in HTML and CSS files
    • Self-assesment. How many points should you get based on the evaluation criteria?
    • Site address.
